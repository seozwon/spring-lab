static 밑에 hello-static 이라는 html 파일을 새로 생성하였습니다.

'localhost:8080/hello-static.html' 하면 이 페이지가 그대로 나옵니다.


동작 방식은 다음과 같습니다.
1. 웹 브라우저에서 localhost:8080/hello-static.html 요청
2. 내장 톰캣 서버가 그 요청을 받아서 스프링한테 물어봄
3. 스프링은 스프링 컨테이너에 hello-static 이라는 컨트롤러가 있는지 찾아봄
3-1. 컨트롤러의 우선순위가 높다.
4. 스프링 : Hello 라는 컨트롤러는 있어(hello1 에 매핑되는 HelloController)
근데 static-hello 라는 컨트롤러는 없어
5. 그럼 resources/static 에 있는 hello-static.html 을 찾아
6. 이걸 반환해줌
